gulp-babel

  1. 在專案中打開終端機(安裝gulp-babel、babel裡的套件core、babel裡的套件preset)

1

$ npm install gulp-babel @babel/core @babel/preset-env

  1. 在gulpfile.js新增程式碼

1
2

const gulp = require('gulp');
const babel = require('gulp-babel');

gulp.task('babel', () => {
  return gulp
    .src('./src/js/*.js') 
    .pipe(babel({preset:['@babel/env']})
    .pipe(gulp.dest('./public/js')) 
});

*.js←所有js檔

  1. 在終端機執行指令

1

$ gulp babel

  1. 執行結果會將src裡的所有js檔(ES5、ES6、ES7)編譯後生成到public裡變成穩定的all.js

<script src="js/all.js"></script>
