常用快捷鍵

Ctrl + W 關閉視窗

Ctrl + Z 還原

Ctrl + X 剪下圖層

Ctrl + C 複製

Ctrl + V 貼上

Ctrl + S 存檔

Ctrl + J 拷貝圖層

Ctrl + shift + S 另存新檔

Ctrl + shift + I 反轉選取

Z 放大鏡

X 前景色/背景色

C 裁切工具

V 移動工具

B 筆刷工具

A 路徑選取工具

S 仿製印章

D 滴管工具

F 快速選取工具

Q 鋼筆工具

注意小細節

  1. 使用濾鏡效果時，要將圖層先轉為智慧型物件。
  2. 裁切工具→刪除裁切的像素(若沒勾選)→拖拉出新範圍，底層是透明格。
  3. 使用橡皮擦工具，等於使用背景色上色。
  4. 使用仿製印章，可以新增圖層，樣本選取"目前及底下的圖層"。
  5. 選擇圖層，點選底下[fx]可以套用陰影、內外框等樣式。
  6. 編輯→透視彎曲，可以設圖釘點彎曲物件，按住Alt點擊刪除圖釘。
  7. 調整面板→色相/飽和度→點選手指再對圖片Ctrl+左鍵拖拉調整色相，左鍵拖拉調整飽和度。

刪除照片中不要的內容

  1. 汙點修復工具(快速處理細小部分)
  2. 填滿內容感知(快速處理大範圍)
  3. 仿製印章(處理較複雜的部份)
