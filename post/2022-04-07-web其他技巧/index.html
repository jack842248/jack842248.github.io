API

  • API：已經寫好程式語言的應用程式介面(Application Programming Interface)
  • Google Maps：地圖導覽
  • Heroku：主機服務
  • Firebase：資料庫儲存

XSS

  • XSS：跨站指令攻擊(Cross-site scripting)
  • 若有輸入框或動態新增的部分
  • 少用innerHTML語法
  • 改用createElement、appendChild

尋找網頁的IP位置

  1. 用chrome打開網頁
  2. 右鍵檢查
  3. Network
  4. Header
  5. Request URL：網址(DNS)
  6. Remote Address：IP位置

查詢ajax是否跨網域

  1. 複製ajax網址
  2. 使用test-cors.org
  3. 貼上ajax網址
  4. XHR start：0 (不支援跨網域)
  5. XHR start：200 (支援跨網域)

chrome搜尋技巧

網頁準則

  • 手機最佳觸控元件長寬為44px
  • 手機最佳非觸控元件長寬為30px
  • 手機瀏覽可省略內文和縮圖
  • 手機瀏覽沒有hover事件
  • 圖片盡量600px以上
  • 在safari瀏覽器上height:100%會跑版
  • logo左上角
  • 主選單在上方
  • 重要訊息出現在未滾動視窗範圍
  • 不超過兩種字型
  • 行距建議1.5倍
  • 字體不小於11px

Google Search Console優化

  1. 登入Google Search Console
  2. 將網址填入後建立好
  3. 到XML Sitemaps.com輸入網址，並下載檔案
  4. 將sitemaps.xml放在根目錄
  5. 將Google Search Console的sitemap填入sitemaps.xml
  6. 檢查：在搜尋引擎上site:網址
  7. 若成功就會出現該網站
  • Social 社交網站
  • Referral 點擊網址
  • Drect 直接輸入
  • Organic Search 搜尋引擎
