元件(Component)

  • 元件的資料都是獨立的，可以重複使用，也可以用元件包元件。

  • 外層使用 props 綁定資料到內層，會即時更新資料。

  • 內層資料有更動時才會觸發 emit event 將傳資料到外層。

template直接建立在app裡面。

  1. 使用 components 建構元件，先自訂"標籤名稱"。

  1. 使用[data] function(){return} 方式傳入資料。

  1. 使用[props]，以[陣列]方式傳入屬性值。

  1. 使用[template]直接建立元件架構。

  1. 在HTML使用"標籤名稱"將元件渲染出來。

  1. 並帶入相對應的值。

var app = new Vue({
  el: "#app",
  data:{
    a:[
      {name:"蘋果",amount:100},
      {name:"香蕉",amount:300},
      {name:"鳳梨",amount:500}
    ]
  },
  components:{
    "tr-component":{
      data: function(){
        return{
        a:[
          {name:"蘋果",amount:100},
          {name:"香蕉",amount:300},
          {name:"鳳梨",amount:500}
        ]
        }
      },
      template:
      `
      <tr>
        <td>{{item.name}}</td>
        <td>{{item.name}}</td>
      </tr>
      `,
      props:["item"]
    }
  }
});

  1. 在HTML使用"標籤名稱"將元件渲染出來。

// <-----html----->
#app
  button(@click="incrementTotal") 按鈕
  p {{cash}} 
  button-counter(v-on:increment="incrementTotal")

// <-----js----->
var app = new Vue({
  el: "#app",
  data:{
    cash: 300
  },
  methods:{
    incrementTotal: function(newNumber){
      this.cash = this.cash + newNumber;
    }
  },
  components:{
    "buttonCounter":{
      data: function(){
        return {
          counter: 1
        }
      },
      template:
      `
      <div>
        <button @click="incrementCounter">增加 {{ counter }}</button>
        <input type="number" v-model="counter"></input>
      </div>
      `,
      methods:{
        incrementCounter: function(){
          this.$emit('increment', Number(this.counter))
        }
      }
    }
  }
});
