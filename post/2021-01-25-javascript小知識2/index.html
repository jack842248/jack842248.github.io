運算式是透過運算子(加減乘除)來得到結果，運算子之間執行方式分為以下兩種:

優先性(Precedence)

  • 優先性決定順序，優先序大的先。

相依性(Associativity)

  • 當遇到優先序相同時，相依性決定執行方向。
優先序 執行方向 符號
20 無 ( )
16 從右至左← ! 、 ++ 、 – 、 typeof 、 delete
14 從左至右→ * 、 / 、 %
13 從左至右→ + 、 -
11 從左至右→ < 、 <= 、 > 、 =>
10 從左至右→ == 、 != 、 === 、 !==
6 從左至右→ &&
5 從左至右→ ll
4 從右至左← … ? … : …
3 從右至左← = 、 += 、 -=
1 從左至右→ ,

/** 範例一 **/
var a = 2 * 2 + 2 * 3;
console.log(a);  //10

//2 * 2 = 4;
//2 * 3 = 6;
//4 + 6 = 10;

/** 範例二 **/
var a = 1 < 2 < 3;
console.log(a);  //true

//1 < 2 = true;
//true < 3 = true; 

/** 範例三 **/
var a = 3 > 2 > 1;
console.log(a);  //false

//3 > 2 = true;
//true > 1 = false;
//因為true==1，所以1不會大於1

運算子中的相等有兩種 :

嚴格相等(Strict Equals)

  • 值相同，型別相同，才能相等。(範例一)

寬鬆相等(Loose Equals)

  • 只要值相同就會相等。(範例二)
  • 布林值、字串會以Number()方式比對。(範例三)
  • null、undefined不能拿來做比對。(範例四)
  • 物件與非物件相比時，以包裹方式做比對。(範例五)
值 包裹方式 結果 值 包裹方式 結果
“1” Number() 1 [1] Number() 1
“2” Number() 2 [“1”] String() “1”
true Number() 1 {a:1} String() Object
false Number() 0 {a:“1”} String() Object
null Number() 0
undefined Number() NaN

/** 範例一 **/
console.log(1 === 1);  //true
console.log(1 === "1");  //false
console.log(1 === true);  //false
console.log(NaN === NaN);  //false(這是例外案)
console.log(+0 === -0);  //true(這是例外案)

/** 範例二 **/
console.log(1 == 1);  //true
console.log(1 == 2);  //false

/** 範例三 **/
console.log(1 == "1");  //true
console.log(1 == true);  //true
console.log(0 == false); //true
console.log("1" == true);  //true
console.log(1 == "true");  //false
console.log("1" == "true");  //false

/** 範例四 **/
console.log(null == 0);  //false
console.log(undefined == 0)  //false
console.log(null == undefined);  //true(這是例外案)

/** 範例五 **/
console.log(1 == [1]);  //true
console.log("1" == [1]);  //true
console.log(1 == ["1"]);  //true
console.log("1" == ["1"]);  //true
console.log({a:1} == {a:1});  //false(物件不能拿來比對)

假值 Falsy false 、 0 、 "" 、 NaN 、 null 、 undefined

運算子中有三種邏輯運算子 :

邏輯運算子 英文 中文 說明
&& AND 而且 左邊是假值(false)回傳結果，是真值(true)回傳右邊
ll OR 或者 左邊是真值(true)回傳結果，是假值(false)回傳右邊
! NOT 不 真值(true)變假值(false)，假值(false)變真值(true)

javascript屬於動態型別，在執行階段才會確定型別，而型別分為以下兩種:

原始型別(Primitive Data Type)

  • 是一個值，沒有屬性

物件型別(Object Type)

  • 以包裹方式建立的型別
型別 typeof 包裹方式
字串 String new String()
數字 Number new Number()
布林值 Boolean new Boolean()
物件 Object
未定義 Undefined
空值 Null
函式 function
資料型態 Symbol
物件實體 Biglnt

當變數在執行階段被賦予值時，可能會轉換型別。而轉換分為以下兩種:

顯性轉換(Explicit Conversion)

  • 直接賦予值
    • 字串 = 數字(範例一)
    • 數字 = 字串(範例二)

  • 透過運算子來轉換型別
    • 字串 * 數字 = 數字(範例一)
