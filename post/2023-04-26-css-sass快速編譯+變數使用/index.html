匯入@import

在all.scss使用@import再彙整出all.css

1
2
3
4
5

@import "variable"; /*常用變數*/
@import "reset"; /*css reset*/
@import "layout"; /*共同框架，第一層基本架構*/
@import "button"; /*按鈕元件*/
@import "資料夾名稱/檔案名稱";

檔名前面加上底線_可以被編譯，但不會被輸出

src/
├─── scss/
│     │ 
│     ├─── _custom.scss  #sass編譯時不會被輸出
│     │     
│     └─── all.scss
│
└─── index.html
