gulp-nunjucks-render

  1. 在專案中打開終端機安裝gulp-nunjucks-render套件

1

$ npm install gulp-nunjucks-render

  1. 在gulpfile.js新增程式碼

 1
 2
 3

const gulp = require('gulp');
const nunjucksRender = require('gulp-nunjucks-render');

gulp.task('layout',function(){
  return gulp
    .src('src/**/*.html')
    .pipe(nunjucksRender({
      path: ['src/templates']
    })) 
    .pipe(gulp.dest('dist'))
    .pipe(browserSync.stream());
});

  1. 在終端機執行指令

1

$ gulp layout

  1. 執行後會自動壓縮img裡的圖片，到dist裡的img資料夾裡
