關於印刷

  1. 開版印刷(印刷廠)
    • 獨立版印刷、凸版印刷(色差小、價格貴)
    • 合版印刷(色差大、價格便宜) ex.健豪、藍格
  2. 數位印刷(超商、印表機)

一般印刷紙材

  1. 名片標準尺寸90x54mm(建議出血3mm)
  2. 中文字體不小於8pt、英文字體不小於6pt
  3. 筆畫線條不小於0.25pt
  4. 注意版本，一般印刷廠軟體版本較舊
  5. 完稿後將文字和線條轉外框
  6. 連結圖檔記得連同附上
  • 珠光紙
    • 表面珍珠般光澤質感
    • 以塑膠材質製成、表面防水、不易撕破
    • 印刷色彩鮮豔
    • 適用於標籤貼

  • 合成紙
    • 表面有霧面質感
    • 以塑膠材質製成、表面防水、不易撕破
    • 適用於大型海報、橫布條、壁面輸出

  • 炫光紙
    • 表面珍珠般光澤感
    • 適合高級印刷、特殊作品

  • 安格紙
    • 表面蛇皮般方格紋路
    • 適合雅痞風格名片印刷

  • 象牙卡
    • 表面平滑但有紙張纖維粗糙質感
    • 色澤沉穩、吸墨性好
    • 可燙金、書寫
    • 頂級象牙卡適合打凸或打凹噴印黑色會呈黑灰色
    • 適合柔和細緻印刷、插畫作品

  • 萊妮紙
    • 表面有不規則十字壓紋
    • 進口美術紙類
    • 吸墨性強、印刷顏色較深
    • 適合典雅風格、質感印刷、插畫作品

  • 一級卡
    • 有分一級卡(薄)、頂級卡(中)、瑞典一級卡(厚)
    • 紙張呈米黃色，平滑有光澤
    • 瑞典一級卡可來製作包裝
    • 適合吊卡、明信片

  • 銅版紙
    • 又分特銅、雪銅、銅西
      • 特銅：光澤度較高、更平滑(傳單、書籍、海報、月曆)
      • 雪銅：霧光效果、不反光、不易穿透(明信片、喜帖、邀請卡)
      • 銅西：西卡紙上塗料，紙張粗、硬度高(書籍封面、卡片、書籤、型錄)
    • 單銅、雙銅則是在單面或雙面上塗料
    • 成本最低，適用於貼紙、小張輸出

  • 模造紙
    • 不上膜、不防水、表面粗糙容易吸墨、油墨不易暈開
    • 適合打凸、燙金
    • 多呈現在環保產品包裝外盒、復古質感

  • 牛皮紙
    • 不上膜、不防水、紙張褐色、易吸墨
    • 可直接書寫或蓋章
    • 復古質感

  • 透明麗龍(透明紙材)
    • 防水、耐磨抗刮、不易破
    • 可燙金或印白墨
    • 多用於產品貼標、LOGO、標準字

  • 後加工處理
    • 上亮膜：表面光滑，會反光
    • 上霧膜：提升質感，不會反光
    • 局部上光：讓重點部分微微凸起、更鮮艷、更光亮建議字體大於9pt、線條大於0.75pt
    • 絲絨加工：頂級卡＋後加工處理，表面絲絨質感、更滑順
    • 燙金：需開版，字級、線條不可太細小
    • 軋型：對紙張進行裁型，刀模線離邊距至少5mm
      • 刀版：裁切線
      • 線版：折線

大圖輸出紙材

  1. 大圖輸出機不適合噴印紋路材質紙張
  2. 水性墨適合室內；油性墨適合室外
  3. 建議圖檔至少要300dpi以上
  4. 上亮膜或霧膜之後顏色會變深
  5. 後加工上亮膜或霧膜，可增加防水性
  6. 通常輸出寬度介於120~150公分

  • 海報紙
    • 非合成塑料材質、紙張純白
    • 因吸墨性佳、容易導致色彩飽和度流失
    • 紙張厚實不易破損
    • 適合印刷打樣、短期室內海報

  • PP相紙(合成紙)
    • 紙張本身偏米黃色
    • 噴印色彩比海報紙鮮亮
    • 有分背膠或無背膠
    • 長期海報、年曆、裱框畫、店面橫幅布條

  • PVC
    • 有分背膠或無背膠
    • 具延展性不易殘膠、防水、耐候性質
    • 有分高遮蔽、灰膠、半透、全透
    • 玻璃貼、人形立牌、電梯門、廣告看板、壁面輸出

板材

  1. 合成板(防水、重量輕)

    • 黏貼貼紙後可再撕下
    • 有分厚度1、2、3、4、5mm
    • 適用於割型用之人形立牌、POP、吊掛圖
    • 受到撞擊容易凹痕

  2. 豪卡板(厚實較硬、不防水、重量輕)

    • 表面有一層紙模，一旦黏貼後不可再撕下
    • 有分厚度1、2、3、4、5mm
    • 適用於裁切方形立牌與會場展場背板
    • 受到撞擊容易凹痕

  3. 發泡板(厚實較硬、不防水、重量輕)

    • 表面有一層紙模，一旦黏貼後不可再撕下
    • 有分厚度1、2、3、4、5mm
    • 適用於裁切方形立牌與會場展場背板
    • 受到撞擊容易凹痕

  4. PP板(側面格狀)

    • 適用於在室內、室外各式廣告看板上
    • 受到撞擊容易凹痕

才數計算

【長cm】×【寬cm】÷ 900 = 才數

範例：150 x 420 ÷ 900 = 70

1才多少錢？ 價錢 ÷ 才數 = 1才價錢

範例：1400 ÷ 70 = 20

30cm × 30cm = 1才
