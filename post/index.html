  p-cp 
    li 香蕉
  
//．蘋果

// <-----js----->
var app = new Vue({
  el: "#app",
  components:{
    "p-cp":{
      template: 
      `
      <ul>
        <li>蘋果</li>
      </ul>
      `
    }
  }
});

  • 使用<template>自己標籤不被輸出，所以不會覆蓋標籤。

  • <slot>可以讓"蘋果"被替換，若沒有被替換，“蘋果"也會變成預設值而顯示出來。

 發表於 2021-02-08  |  1 分鐘  |  30 個字

生命週期

英文 中文 說明
開始創建
beforeCreate 準備載入資料
數據觀測建立
created 載入資料 要載入ajax要在這階段之後
編譯模板
beforeMount 準備繪製元件
模板建立完成
mounted 繪製元件 要操作DOM元素要在這階段之後
資料變動
updated 更新資料時觸發
activated 有<keep-alive>時才會觸發
deactivated 有<keep-alive>時才會觸發
beforeDestroy 準備移除資料
移除觀測、子元件、監聽事件
destroyed 移除資料
