## 標題
開頭用`#`撰寫，分別對應的是`<h1>`到`<h6>`標題
|名稱|md寫法|對應html|
|---|---|---|
|一級標題|`# 文字內容`|`<h1>文字內容</h1>`|
|二級標題|`## 文字內容`|`<h2>文字內容</h2>`|
|三級標題|`### 文字內容`|`<h3>文字內容</h3>`|
|四級標題|`#### 文字內容`|`<h4>文字內容</h4>`|
|五級標題|`##### 文字內容`|`<h5>文字內容</h5>`|
|最小標題|`###### 文字內容`|`<h6>文字內容</h6>`|

-----------------------------------------------
## 文字
用`*`來包覆，分別是斜體、粗體、斜粗體
|名稱|md寫法|對應html|
|---|---|---|
|斜體|`*文字內容*`|`<em>文字內容</em>`|
|粗體|`**文字內容**`|`<strong>文字內容</strong>`|
|斜粗體|`***文字內容***`|`<strong><em>文字內容</em></strong>`|

-----------------------------------------------
## 文字連結
`[]`內是顯示文字，`()`是`href`，但是`md`無法另開新視窗
```
[Google連結](https://www.google.com.tw/)
```
對應html:
```
<a href="https://google.com">https://google.com</a>
```

-----------------------------------------------
## 圖片
`[]`對應的是`alt`，`()`對應的是`src`
```
![圖片](/img/path.jpg)
```
對應html:
```
<img src="/img/path.jpg" alt="圖片">
```

-----------------------------------------------
## 刪除文字
文字用2個`~`來包覆是刪除線(但不是所有`Markdown`引擎都支援)
|名稱|md寫法|對應html|
|---|---|---|
|刪除線|`~~文字內容~~`|`<del>文字內容</del>`|
-----------------------------------------------
## 有序序列
`數字`＋`.` 排序下來是有序序列，對應的是`<ol>`
```
1. 有序序列
2. 有序序列
3. 有序序列
```
對應html:
```
<ol>
    <li>有序序列</li>
    <li>有序序列</li>
    <li>有序序列</li>
</ol>
```

-----------------------------------------------
## 無序序列
用`*`排序下來是無序序列，可以往內縮排寫到第二層、第三層
```
* 無序序列
* 無序序列
* 無序序列
```
對應html:
```
<ul>
    <li>無序序列</li>
    <li>無序序列</li>
    <li>無序序列</li>
</ul>
```

-----------------------------------------------
## 引言
用`>`排序下來是引言，對應的是`<blockquote>`
```
> 第一層引言
>> 第二層引言
>>> 第三層引言
```
對應html:
```
<blockquote>第一層引言
    <blockquote>第二層引言
        <blockquote>第三層引言
        </blockquote>
    </blockquote>
</blockquote>
```

-----------------------------------------------
## 勾選清單
```
- [ ] 未勾選清單
- [x] 已勾選清單
```
對應html:
```
<ul>
  <li>
    <input type="checkbox" disabled>
    未勾選清單
  </li>
  <li>
    <input type="checkbox" checked disabled>
    已勾選清單
  </li>
</ul>
```

-----------------------------------------------
## 定義型清單
```
名詞
: 說明
```
對應html:
```
<dl>
  <dt>名詞</dt>
  <dd>說明</dd>
</dl>
```

-----------------------------------------------
## 表格
```
|表頭一|表頭二|表頭三| 
|---|---|---| 
|內容一|內容二|內容三|
|內容四|內容五|內容六|
```
對應html:
```
<table>
  <thead>
    <tr>
      <th>表頭一</th>
      <th>表頭二</th>
      <th>表頭三</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>內容一</td>
      <td>內容二</td>
      <td>內容三</td>
    </tr>
    <tr>
      <td>內容四</td>
      <td>內容五</td>
      <td>內容六</td>
    </tr>
  </tbody>
</table>
```
若要讓整列置左：`:---`
若要讓整列置中：`:---:`
若要讓整列置右：`---:`

## 程式碼片段
|名稱|md寫法|對應html|
|---|---|---|
|單行程式碼|(反引號)程式碼(反引號)|`<code></code>`|
|多行程式碼|(三個反引號)文字內容(三個反引號)|`<pre><code></code></pre>`|

-----------------------------------------------
## 分隔線
以下三種寫法都會產生分隔線
|md寫法|對應html|
|---|---|
|`---`|`<hr>`|
|`***`|`<hr>`|
|`___`|`<hr>`|

## 跳脫字元
若在最前方加上`\`，就不會是該對應標籤
```
\* 不是斜體
\# 不是標題
```
